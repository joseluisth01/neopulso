2. Finalidad del tratamiento

Tratamos tus datos personales para las siguientes finalidades:

  • Gestionar y responder a las solicitudes de información y contacto enviadas a través del formulario web.
  • Enviar información comercial sobre nuestros servicios, si así lo has autorizado.
  • Cumplir con las obligaciones legales que nos sean aplicables.
  • Llevar estadísticas de visitas y comportamiento de navegación para mejorar el sitio web (con tu consentimiento).

4. Plazo de conservación

Conservamos tus datos durante el tiempo necesario para cumplir la finalidad para la que fueron recogidos. Los datos de contacto se conservan durante 3 años desde la última interacción. Transcurrido dicho plazo, los datos serán suprimidos o anonimizados.

5. Destinatarios de los datos

No cedemos tus datos a terceros salvo obligación legal. Podemos compartir datos con proveedores de servicios que actúan como encargados del tratamiento (hosting, plataforma de email), con quienes tenemos suscritos contratos de encargo que garantizan el cumplimiento del RGPD.

6. Tus derechos

Puedes ejercer los siguientes derechos dirigiéndote a info@neopulso.es:

  • Acceso: conocer qué datos tuyos tratamos.
  • Rectificación: corregir datos inexactos o incompletos.
  • Supresión: solicitar la eliminación de tus datos.
  • Oposición: oponerte al tratamiento con fines comerciales.
  • Limitación: solicitar la restricción del tratamiento.
  • Portabilidad: recibir tus datos en formato estructurado.

También tienes derecho a presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD).

7. Cookies

Este sitio web utiliza cookies. Para más información, consulta nuestra Política de Cookies.

8. Seguridad

Aplicamos medidas técnicas y organizativas adecuadas para proteger tus datos frente a accesos no autorizados, pérdida o alteración, incluyendo cifrado HTTPS, controles de acceso y copias de seguridad periódicas.

Para cualquier consulta sobre esta política, escríbenos a info@neopulso.es.
